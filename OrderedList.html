1. HTML :-This tag is the root element of an HTML document.
  2. head :-Element contains metadata and information about the document
  3. title :-Tag defines the title of the HTML document
  4. meta :- Tag Provide metadata about the HTML Document
  5. link :- Tag is used to link external resources like stylesheet, favicon, web fonts HTML Documents
  6. body :-Element contains the visible content of the webpage.
  7. p :-This tag defines a paragraph of text
  8. a :-The anchor tag is used to create hyperlinks
  9. img :- The tag embeds images in the webpage
  10. table :-The tag defines a table
